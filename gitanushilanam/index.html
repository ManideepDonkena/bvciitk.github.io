Bhagavad Gita Essay Competition

Submission Type : Document file in pdf format
Submission Link : Submit Here

Instructions :

  • The participant can write on anyone from the above-given topics
  • Only one entry per person will be accepted
  • The content written in the blog should be your own. If plagiarism is found, then the entry would be disqualified
  • Give it a snappy, interesting titl
  • Kindly mention your Full Name, Age, Mobile number at the beginning of the file you are submitting

Size Limit:

  • 6-10 Years: ≤ 200 words
  • 10-14 years: ≤ 250 word
  • 14-18 years: ≤ 350 words
  • Adults(18+): ≤ 500 wordss

निर्देश :

  • प्रतिभागी दिए गए विषयों में से किसी पर भी लिख सकता है
  • केवल एक व्यक्ति से एक ही प्रविष्टि को स्वीकार किया जाएगा
  • ब्लॉग में लिखा कंटेंट आपका अपना होना चाहिए। यदि साहित्यिक चोरी पाई जाती है, तो प्रविष्टि को अयोग्य घोषित कर दिया जाएगा
  • इसे एक तेज़, दिलचस्प शीर्षक दें
  • आप जिस फाइल को सबमिट कर रहे हैं, उसकी शुरुआत में कृपया अपना पूरा नाम, उम्र, मोबाइल नंबर का उल्लेख करें

निबंध का आकार:

  • 6-10 वर्ष: ≤ 200 शब्द
  • 10-14 वर्ष: ≤ 250 शब्द
  • 14-18 वर्ष: ≤ 350 शब्द
  • वयस्क (18+): ≤ 500 शब्द
